## Идеи на будущее

### 1. «Точно хотите подсмотреть?»
Если уходить с теста на теорию — диалоговое окно: «Точно хотите подсмотреть, а не проверить знания?». Если да — в конце теста пометка «Пройдено, но с читерством». Диалог срабатывает только в первый раз за тест.

**Сложность:** лёгкая (boolean в quiz store). **Когда:** неделя 4 (геймификация).

### 2. Марафон вопросов
Отдельная вкладка «Марафон» — 250 вопросов по собеседованиям. Рандомная выборка из всех уроков, таймер, статистика по темам.

**Сложность:** средняя. **Когда:** неделя 5-6 (spaced repetition).

### 3. AI-ментор
Личный ментор внутри приложения, который готовит к собеседованиям. Два режима:

- **Текстовый:** вопрос → ответ → комментарий/уточнение. Реализация: Claude API + streaming.
- **Голосовой:** вопрос голосом → отвечаешь голосом → фидбэк. Реализация: Web Speech API (распознавание) + TTS (синтез). Тренирует именно устный навык ответа.

**Сложность:** высокая (отдельный мини-проект). **Когда:** неделя 7-8 как «wow-фактор» для портфолио.

### Разбор затыка: зачем уменьшать запросы к БД

Было 3 запроса (SELECT → INSERT/UPDATE → UPDATE user) — каждый это round-trip через сеть (~50мс). При одном юзере незаметно, но при 100 юзерах × 10 действий = разница между 500мс и 150мс. На собесе спрашивают не «работает ли?», а «масштабируется ли?».

## Продукт: мысли о развитии

### Почему это может стать продуктом

На рынке труда сейчас много людей в такой же ситуации: мидл-фронтенд, хочет расти, готовится к собесам, борется с саботажем. Приложение закрывает реальную боль, которую я испытываю сама — это лучшая стартовая точка для продукта. «Я сделала это для себя, потом открыла другим» — классический путь инди-продуктов.

### Преимущества Telegram Mini App как платформы

- **Нулевой барьер входа** — не нужно скачивать из стора, просто открыл бота
- **Telegram Stars** — встроенные платежи без 30% комиссии Apple/Google
- **Вирусность** — бота можно переслать, кинуть в чат, поделиться ссылкой
- **Push через бота** — уведомления «пора повторить» без разрешений на нотификации

### Если захочу на Android/iOS

| | Google Play | App Store |
|---|---|---|
| Плата | $25 разово | $99/год |
| Технология | React Native — переиспользую ~60-70% кода (stores, API, типы). Но UI заново (`<View>` вместо `<div>`) |
| Комиссия | 15-30% с платежей | 15-30% с платежей |

### Модели монетизации

1. **Freemium** — базовые уроки бесплатно, продвинутые за подписку ($3-10/мес). Самая популярная (Duolingo, LeetCode).
2. **Одноразовая покупка** — полный доступ за $5-15. Проще, но нет повторяющегося дохода.
3. **Telegram Stars** — продажа контента через встроенные платежи Telegram. Без комиссий сторов. Наше преимущество.

### План (по шагам, не фантазировать)

```
Сейчас:      Добить проект до 15-20 уроков, пользоваться самой
Месяц 2:     Открыть бота публично, кинуть в чаты фронтендеров
Месяц 3:     Посмотреть метрики — кто заходит, что проходят, где бросают
Потом:        Решить — монетизировать или оставить бесплатным для портфолио
```

### Важно помнить

Для **поиска работы** ценность проекта уже есть — «спроектировала, реализовала и задеплоила fullstack-приложение с аутентификацией, БД и реальными пользователями». Это мидл+ уровень. Монетизация — бонус, не цель на данном этапе. Сначала работа и стабильный доход, потом эксперименты с продуктом.
